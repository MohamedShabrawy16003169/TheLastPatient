# The Last Patient

The Last Patient is a first-person psychological horror game developed in **Unity 6** as a university project.

The game focuses on creating an immersive atmosphere through environmental storytelling, dynamic lighting, sound design, and interactive gameplay. Players explore a mysterious environment, uncover clues, solve puzzles, and experience a narrative-driven horror adventure.

## Features

* First-person exploration
* Psychological horror atmosphere
* Interactive objects and puzzles
* Dynamic lighting and audio effects
* Environmental storytelling
* Unity 6 project structure

## Technologies Used

* Unity 6
* C#
* Universal Render Pipeline (URP)

## Getting Started

1. Clone the repository.
2. Open the project using Unity 6.
3. Allow Unity to import all assets.
4. Open the main scene and press **Play**.
